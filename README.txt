================================================================================
КРИПТОГРАФІЧНЕ ЗАВДАННЯ - ШИФРУВАННЯ ТЕКСТУ ОСКАРА ВАЙЛЬДА
================================================================================

Цей проєкт реалізує три класичні криптографічні шифри для шифрування
тексту з передмови до "Портрета Доріана Ґрея" Оскара Вайльда.


ШВИДКИЙ СТАРТ
================================================================================

1. Запустити програму:
   python3 cryptography.py

2. Інтерактивний режим:
   python3 interactive_example.py


ФАЙЛИ ПРОЄКТУ
================================================================================

ОСНОВНІ ПРОГРАМИ:
  cryptography.py                - Головна програма з усіма шифрами
  interactive_example.py         - Інтерактивні приклади

ДЛЯ ЗВІТУ:
  ЗВІТ_ДЛЯ_GOOGLE_DOCS.txt      - Текст звіту (скопіювати в Google Docs)
  ІНСТРУКЦІЯ_ДЛЯ_СКРІНШОТІВ.txt - Як зробити скріншоти для звіту

ДОВІДКА:
  ЗВІТ.txt                       - Детальний звіт
  EXAMPLES.md                    - Приклади з поясненнями
  README.txt                     - Цей файл

РЕЗУЛЬТАТИ (створюються після запуску):
  vigenere_encrypted.txt         - Шифр Віженера
  transposition_encrypted.txt    - Проста перестановка
  double_transposition_encrypted.txt - Подвійна перестановка
  tabular_encrypted.txt          - Табличний шифр
  combined_encrypted.txt         - Комбінований шифр


РЕАЛІЗОВАНІ ШИФРИ
================================================================================

1. ШИФР ВІЖЕНЕРА (Vigenère Cipher)
   Рівень 1: Шифрування з ключем CRYPTOGRAPHY
   Рівень 2: Криптоаналіз (метод Касіскі + тест Фрідмана)

2. ШИФР ПЕРЕСТАНОВКИ (Transposition Cipher)
   Рівень 1: Проста перестановка з ключем SECRET
   Рівень 2: Подвійна перестановка (SECRET + CRYPTO)

3. ТАБЛИЧНИЙ ШИФР (Route Cipher)
   Рівень 1: Табличний шифр з ключем MATRIX
   Рівень 2: Комбінований шифр (Віженера + Табличний)


ЯК ЗРОБИТИ ЗВІТ
================================================================================

1. Запустіть програми та зробіть скріншоти
   (див. ІНСТРУКЦІЯ_ДЛЯ_СКРІНШОТІВ.txt)

2. Відкрийте ЗВІТ_ДЛЯ_GOOGLE_DOCS.txt

3. Скопіюйте текст у Google Docs

4. Вставте скріншоти на позначені місця

5. Готово!
